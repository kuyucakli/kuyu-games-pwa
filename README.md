# Kuyu Games

A minimalist progressive web app (PWA) featuring physics-based casual games.

The current version includes two games: **Tahterevallis** and **Tilt**(under development).

This is an experimental project focused on core gameplay systems rather than content completeness.

Supabase is currently used for authentication and will later support progress persistence and leaderboards.

[You can use live app here. ](https://kuyu-games-pwa.vercel.app)

## Games

### Tahterevallis

A physics-driven game focused on balance and controlled motion.

### Tilt (under development)

A tilt-based game emphasizing precision and timing.

## Features

- Physics-driven ball mechanics
- Level-based progression
- Hand-painted watercolor backgrounds
- Audio
- Simple particle fx

## Tech Stack

- Supabase auth
- TypeScript
- Next.js
- Three.js
- Rapier (wasm)
- Vite
- Vitest

## Getting Started

```bash
pnpm install
```
